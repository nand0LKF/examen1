¿Qué es el Padding?

  • Espacio interno entre el contenido y el borde del elemento
  • A diferencia del margin (espacio externo)
  • Afecta directamente el tamaño total del elemento
  • No se colapsa como los márgenes

La Regla del Reloj (TRBL)

  • Orden de los valores: Top → Right → Bottom → Left
  • Sentido horario a partir de las 12
  • Aplica para padding y margin
  • Mnemotecnia: "El reloj nunca miente"
/* Sentido horario */
padding: 10px 20px 15px 25px;
/*
10px = Top
20px = Right
15px = Bottom
25px = Left
*/

Sintaxis Abreviada

  • 1 valor: todos los lados
    (padding: 20px)
  • 2 valores: vertical + horizontal
    (padding: 10px 30px)
  • 3 valores: top + horizontal + bottom
    (padding: 10px 20px 15px)
  • 4 valores: TRBL completo
    (padding: 5px 10px 15px 20px)

El Problema Matemático: Box Sizing

  • content-box (default):
    ancho = contenido + padding + borde
  • border-box (recomendado):
    ancho = contenido (incluye padding y borde)
  • Solución moderna:
    *, *::before, *::after {
      box-sizing: border-box;
    }

Ejemplos Prácticos: Botones

  • Padding controla el "aliento" del botón
  • Mejor legibilidad y área de clic
  • Consistencia visual con diferentes textos
  • Nunca uses height/width en botones, usa padding
.btn-small { padding: 5px 10px }
.btn-medium { padding: 10px 20px }
.btn-large { padding: 15px 30px }
6/8

Puntos Clave

  • Padding = espacio interno, margin = espacio externo
  • TRBL: Orden de valores en sentido horario
  • Sintaxis abreviada ahorra líneas de código
  • Siempre usa box-sizing: border-box globalmente
  • Padding es la herramienta ideal para componentes como botones
